scala知识总结

语法基础

与java的区别

1.避免以’$’或者’_’结尾
2.yield为Scala保留关键字
3.类名首字母大写，如果由多个单词拼接而成，每个单词首字母大写

换行符

‘;’是可选的，但如果一行里面有多条语句，那么换行符是需要的。例如 val s = “”;println(s)

数据类型

和java类型相同，不同的是多了几种Any,AnyVal,AnyRef,Null,Nothing.其中java的Object对应于这里的AnyRef.值得注意的是：在Scala中所有的都是对象，即使是Int等值类型也是对象。

val和var

val用来声明常量，var用来声明变量。使用val声明的常量，声明了以后就不能再修改了。这点和java的final相似。使用var声明的变量可以反复赋值。值得注意的是：在scala中，因为支持函数式编程，所以val可以接受多种类型，比如,数组，函数等等.例如：val pl = (40,”Foo”)

访问修饰符

和java类似，缺少了java的default类型。默认情况下，Scala对象的访问级别都是public。而在java中，是default。

1.private限定符，比java更严格，在嵌套类情况下，外层类甚至不能访问被嵌套类的私有成员。

(new Inner).f( ) 访问不合法是因为 f 在 Inner 中被声明为 private，而访问不在类Inner之内。但在 InnerMost 里访问f就没有问题的，因为这个访问包含在 Inner 类之内。
2.protected限定符，只允许保护成员在定义了该成员的类的字类中被访问。区别于java的是，java允许同一个包中的其他类访问
3.public没有限制，在任何地方都可以被访问
4.作用域保护。Scala中，访问修饰符可以通过使用限定词强调。格式为：private[x] 或 protected[x]。这里的x指代某个所属的包，类或单例对象。如果写成private[x],读作“这个成员出了对[…]中的类或[…]中的包中的类以及它们的伴生对象可见外，对其他所有类都是private”。

循环

其中while循环和do…while循环和java类似。重点记录for循环。
1.基础用法。其中，1 to 10表示循环从1到10包括1和10. 即[1,10] <- 表示为变量x赋值。1 until 10表示从1到10不包括10.即[1,10).

2.使用分号’;’来设置多个区间。

3.for循环集合，和java里面的增强for类似。

4.for循环中过滤。Scala可以使用一个或者多个if语句来过滤一些元素。

5.for使用yield。Scala中可以讲for循环的返回值作为一个变量存储。特别的，x这里也可以当作一个表达式来处理。也就是说，可以对x变量进行操作，例如返回 x+1 也是可以的。

functions（函数） 和 methods（方法）

函数式带参数的表达式。例如：（x:Int）=> x+1 其中 => 是参数列表。右边是涉及参数的表达式。
函数也是可以命名的。例如：val addOne = (x:Int) => x + 1;println(addOne(1))
函数可以带有多个参数。例如：val add = (x:Int,y:Int) => x + y;println(add(1,2))
函数也可以不需要参数。例如：val getTheAnswer = () => 42;println(getTheAnswer())

方法使用def定义。def后跟一个名称，参数列表，返回类型和正文。def add(x:Int,y:Int):Int = x + y。
方法使用多个参数。def addThenMultiply(x:Int,y:Int)(multiplier:Int):Int = (x+y)*multiplier;println(addThenMultiply(1,2)(3))

字符串

在Scala中，字符串的类型实际上是Java String，它本身没有String类。String是一个不可变对象，所以该对象不可修改。这就意味着你如果修改字符串就会产生一个新的字符串对象。
声明字符串：var greeting = “hello world” ; var greeting:String = “hello world”;
可修改字符串：val buf = new StringBuilder;buf += ‘a’; buf += “bcdef”;println(“buf is:”+buf.toString);
类比java：常用的字符串方法有：length();concat();chartAt();indexOf();subString();trim();…

集合

Scala集合分为可变和不可变集合。可变集合可以在适当的地方被扩展或者更新。这意味着你可以修改，添加，移除一个集合的元素。而不可变集合，相比之下，永远不可改变。不过，你仍然可以模拟修改，添加，移除或更新操作。这些操作在每一种情况下都返回一个新的集合。同时是原来的集合不发生改变。集合有：

其他常用的方法

  • addString(b:StringBuilder):StringBuilder 和addString(b:StringBuilder，sep:String):StringBuilder

    将列表中的所有元素添加到StringBuilder，可指定分隔符。注意：这里是将列表中的元素添加到StringBuilder，而不是将StringBuilder添加到列表中。

  • apply(n:Int):A:通过索引获取集合中的元素。

  • distinct：List[A] 去除列表中的重复元素，并返回新列表

  • contains(ele:Any):Boolean:检测列表中是否包含指定的元素

  • drop(n:Int):List[A]:丢弃前n个元素，并返回新列表。dropRight(n:Int):List[A]:丢弃最后n个元素

  • take(n:Int):List[A]:获取列表的后几个元素 。takeRight(n:Int):List[A]:提起列表的后n个元素

  • toArray:Array[A]:列表转化为数组 toMap[T,U]:Map[T,U] toSeq:Seq[A] toSet[]:Set[B] toString():String
